# Shvan Haziz CONV

A premium, Netflix-style intelligent file converter for iOS (SwiftUI).

## What it does now (offline)
- Detects file type instantly (extension + UTType)
- Shows only valid conversions (rule-based, deterministic)
- Converts on-device:
  - TXT ↔ RTF
  - TXT/RTF → PDF
  - PDF → TXT (best-effort)
- Calm progress, share/save output
- Conversion history stored locally

## Planned (secure cloud conversion)
- DOC/DOCX/DOCM/DOTX/DOTM → PDF/TXT
- PPT/PPTX/PPTM/POTX/POTM/PPSX/PPSM → PDF/Images/MP4

Macros are never executed.
